MY PROJECTS

A showcase of my Data Science and Machine Learning projects, these works reflect my passion for leveraging data to solve real-world challenges and drive innovation

Game Recommendation System

A user-based game recommendation system built with a simple Neural Network model. This project uses an embedding representation for each game and leverages the similarity of embedding weights to generate relevant game recommendations based on user preferences.

🚀 Go Project 🎥 Demo Video

Chemical Trend Estimate

This project focuses on forecasting the monthly consumption of sodium hypochlorite (NaOCl) for cooling tower treatment. By applying data preprocessing (cleaning, encoding, and feature engineering) and ensemble learning techniques with stacking, I developed a predictive model that estimates chemical usage more accurately. The solution includes a full pipeline and a user-friendly API built with Flask and HTML, helping industries optimize inventory planning and cost management.

🌐 Web App 💻 GitHub
🎥 Demo Video

House Price Prediction

This project aims to build a predictive system for house pricing based on factors such as location, area, facilities, and building conditions. I conducted data preprocessing, feature selection (RFE), and hyperparameter tuning to improve model performance. The project compared multiple models, including Linear Regression, Random Forest, and XGBoost.

🚀 Go Project

Spam Message Detection

A machine learning project that detects spam messages using the SMS Spam Collection Dataset. The workflow includes text preprocessing (tokenization, lemmatization, Word2Vec), model training with Random Forest and XGBoost (optimized via GridSearch), and deployment through a Flask-based user API.

🚀 Go Project 🎥 Demo Video

Fraud Detection

This project focuses on detecting fraudulent transactions in financial data, where fraud cases are rare and challenging to identify due to class imbalance and evolving fraud patterns. I performed data preprocessing such as encoding, scaling, and handling imbalance, then trained and optimized multiple machine learning models (Logistic Regression, Decision Tree, Random Forest, and AdaBoost) using GridSearchCV
